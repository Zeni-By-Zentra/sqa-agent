# Comunidad SQA Agent

Una comunidad abierta de desarrolladores colombianos y globales que creen en software de calidad.

## Código de Conducta

- Respeto mutual ante todo
- Críticas constructivas, nunca personales
- Ayudamos a quien está aprendiendo
- El español y el inglés son bienvenidos

## Cómo participar

1. Usa el agente: Instálalo y pruébalo en tu proyecto
2. Reporta bugs: Abre un issue
3. Propón mejoras: Abre un PR
4. Comparte: Cuéntanos en qué proyecto te ayudó
